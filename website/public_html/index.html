VPLIDE

Online Code Editor for Exams/Assignments

DETAILS

more about VPL and our goals

Virtual Programming Lab (VPL) is an activity module that manages programming assignments. Current VPL module used by METU is not user friendly. The assistants have difficulties in preparing lab sessions, students have a very ugly development environment. Also, the jail server used is not easy to manage and scalable. Aim of the project is to create a visually enhanced, easy to use module and software for online programming exams and lectures.

Execution server will be replaced by a more scalable technology and browser side computation will be used to offload the server tasks. Project requires handling of laboratory sessions with multiple programming languages. Visually rich editor with completion and highlighting should be used.

A protocol between server and browser side should provide compilation, execution and debugging stages in a modern IDE like environment. Various environments, programming languages should be configured by the assistant. The test data and evaluators should be easily prepared. The execution servers should be easily managing as a Kubernetes or similar cluster system. Security is essential, student communication should be avoided. Cheating detection is not a required but good to have feature.
